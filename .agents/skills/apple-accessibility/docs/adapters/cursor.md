# Cursor Adapter

Use this adapter when running these skills in Cursor.

## Use this when

- Auditing SwiftUI/UIKit/AppKit files directly in-editor
- Requesting patch-ready output with minimal back-and-forth

## Skip this when

- You are using non-Cursor agents (use matching adapter instead)

## Jump to

- SwiftUI: `skills/swiftui-accessibility-auditor/SKILL.md`
- UIKit: `skills/uikit-accessibility-auditor/SKILL.md`
- AppKit: `skills/appkit-accessibility-auditor/SKILL.md`

## Request pattern

"Audit this <framework> code with the corresponding accessibility auditor. Return P0/P1/P2 findings, patch-ready changes, and manual verification steps."
